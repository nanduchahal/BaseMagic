# BaseMagic

BaseMagic is an AI Agent built on Base that combines:

- ERC-8004 Agent Identity
- x402 Payment Protocol
- Model Context Protocol (MCP)
- Agent-to-Agent (A2A) Communication

## Current Features

### Identity

- ERC-8004 registered agent
- Public Agent Card
- Railway deployment

### Payments

- x402 middleware
- Automatic HTTP 402 payment challenge
- x402 Buyer Runtime
- Automatic payment retry

### AI

- OpenAI-powered responses
- MCP tool support
- JSON-RPC A2A endpoint

## Current Status

Sprint 1 completed:

- Seller Runtime
- Buyer Runtime
- Automatic x402 payment flow

Upcoming work:

- Streaming payments
- Rich Agent Card
- SIWA
- EAS Reputation

BaseMagic is being built as a production-quality reference implementation for autonomous economic AI agents on Base.
